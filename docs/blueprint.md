# **App Name**: FinSim

## Core Features:

- SIP Calculator: Calculate projected returns for Systematic Investment Plans by inputting investment amount, duration, and expected rate of return.
- SWP Calculator: Calculate potential withdrawals from Systematic Withdrawal Plans based on initial investment, withdrawal amount, and expected rate of return.
- Compound Interest Calculator: Calculate compound interest earnings by providing principal amount, interest rate, compounding frequency, and investment duration.
- Interactive Charts: Visualize projected returns and withdrawals through interactive charts, allowing users to adjust parameters and observe changes in real-time.
- Detailed Result Breakdowns: Present detailed breakdowns of calculation results, including total investment, total interest earned, and withdrawal schedules.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey trust and stability.
- Background color: Light gray (#F0F4F8), providing a clean, professional backdrop.
- Accent color: Teal (#20C2E0) to highlight key interactive elements and financial data.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective look that supports clarity in financial data.
- Use clean, minimalist icons to represent investment parameters and calculation types.
- Employ smooth UI transitions and subtle animations to enhance user experience and provide visual feedback during calculations and parameter adjustments.
